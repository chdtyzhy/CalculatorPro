# CalculatorPro UI 设计规范

## 参考应用
- App Store: Calculator with History (ID1458583388)
- 风格: iOS 原生计算器风格

## 屏幕尺寸基准
- 设计基准: iPhone 14 Pro (393 x 852 points)
- 安全区域: top 47pt, bottom 34pt

## 布局结构

### 1. 显示区域 (Display Area)
- **位置**: 顶部
- **高度**: 120pt (约屏幕高度的 15%)
- **背景**: 黑色 (#000000)
- **内边距**: 左右 20pt, 上下 10pt

#### 显示内容
- **上一行**: 操作预览 (如 "123 +")
  - 字体: SF Pro Regular, 24pt
  - 颜色: 灰色 (#8E8E93)
  - 对齐: 右对齐
- **主显示**: 当前数字
  - 字体: SF Pro Light, 64pt
  - 颜色: 白色 (#FFFFFF)
  - 对齐: 右对齐
  - 最小缩放: 0.3

### 2. 按钮区域 (Button Area)
- **位置**: 显示区域下方
- **高度**: 屏幕剩余高度 - 安全区域
- **背景**: 黑色 (#000000)

#### 按钮布局
- **列数**: 4 列
- **行数**: 5 行
- **按钮间距**: 12pt (水平和垂直)
- **边距**: 左右 16pt, 底部 20pt

#### 按钮尺寸计算
```
可用宽度 = 屏幕宽度 - 左右边距(32pt)
按钮宽度 = (可用宽度 - 间距(36pt)) / 4
按钮高度 = 按钮宽度 (正方形)
```

以 iPhone 14 Pro (393pt) 为例:
- 可用宽度 = 393 - 32 = 361pt
- 按钮宽度 = (361 - 36) / 4 = 81.25pt ≈ 81pt

#### 按钮样式

**数字按钮 (0-9, .)**
- 背景: 深灰色 (#333333)
- 文字: 白色, 32pt, Medium
- 形状: 圆形

**功能按钮 (C, +/-, %, ⌫)**
- 背景: 浅灰色 (#A5A5A5)
- 文字: 黑色, 28pt, Medium
- 形状: 圆形

**运算符按钮 (+, −, ×, ÷)**
- 背景: 橙色 (#FF9F0A)
- 文字: 白色, 36pt, Medium
- 形状: 圆形

**等号按钮 (=)**
- 背景: 橙色 (#FF9F0A)
- 文字: 白色, 36pt, Medium
- 形状: 圆形

**历史按钮**
- 背景: 深灰色 (#333333)
- 图标: clock.arrow.circlepath
- 图标大小: 28pt
- 形状: 圆形

#### 特殊按钮
- **0 按钮**: 占 2 列宽度 (2 × 按钮宽度 + 间距)

### 3. 按钮排列

```
第1行: [C] [+/-] [%] [⌫] [历史]
第2行: [7] [8] [9] [÷]
第3行: [4] [5] [6] [×]
第4行: [1] [2] [3] [−]
第5行: [0    ] [.] [+] [=]
```

## 颜色规范

| 元素 | 颜色 | Hex |
|------|------|-----|
| 背景 | 黑色 | #000000 |
| 数字按钮 | 深灰 | #333333 |
| 功能按钮 | 浅灰 | #A5A5A5 |
| 运算符/等号 | 橙色 | #FF9F0A |
| 主文字 | 白色 | #FFFFFF |
| 次要文字 | 灰色 | #8E8E93 |

## 字体规范

| 用途 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 显示数字 | SF Pro | 64pt | Light |
| 操作预览 | SF Pro | 24pt | Regular |
| 数字按钮 | SF Pro | 32pt | Medium |
| 功能按钮 | SF Pro | 28pt | Medium |
| 运算符 | SF Pro | 36pt | Medium |

## 交互规范

- 按钮点击反馈: 透明度变化 (0.7)
- 动画时长: 0.1s
- 历史记录: 从底部弹出 Sheet

## 适配规则

1. **小屏幕 (iPhone SE)**: 减小按钮间距至 8pt, 字体相应缩小
2. **大屏幕 (iPhone Pro Max)**: 保持按钮大小, 增加间距
3. **iPad**: 居中显示, 最大宽度 400pt
